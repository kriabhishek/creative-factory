# Creative Factory — Replit Ad Design System

All ad mockups generated by `/creative-factory` must follow this design language, derived from Replit's actual product marketing (March 2026).

---

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| `brand-orange` | `#FF3C00` | CTAs, logo, hero backgrounds, accent highlights |
| `dark-bg` | `#0E1525` | Product UI backgrounds, dark-theme ads |
| `dark-surface` | `#1A2233` | Cards, panels, elevated surfaces on dark |
| `dark-border` | `#2A3548` | Borders, dividers on dark backgrounds |
| `light-bg` | `#F5F1EB` | Marketing page backgrounds, testimonial sections |
| `text-primary` | `#FFFFFF` | Headings on dark, text on orange |
| `text-secondary` | `#C8D4E6` | Body copy on dark backgrounds |
| `text-muted` | `#8B9AB5` | Captions, labels, secondary info on dark |
| `text-dark` | `#1A1A1A` | Headings on light/white backgrounds |
| `green-live` | `#28CA41` | Live badges, success states, security indicators |

## Typography

- **Font:** Inter (primary), -apple-system fallback
- **Headlines:** 700-900 weight, large (28-48px in ad units)
- **Body:** 400-500 weight, 16-20px
- **Captions/labels:** 500-600 weight, 12-14px, uppercase tracking for labels
- **Line height:** 1.3 for headlines, 1.5 for body
- **Color hierarchy:** White for headlines on dark, `text-secondary` for body on dark, black for headlines on light

## Layout Patterns

### Pattern 1: Orange Feature Panel (from product pages)
- Left: Orange (`#FF3C00`) background with white headline + body text
- Right: Dark product screenshot with subtle drop shadow, slightly overlapping the orange boundary
- Used for: Feature callouts, capability highlights

### Pattern 2: Dark Product Showcase (from Desktop App page)
- Full dark background (`#0E1525`)
- Product screenshots in dark mode, overlapping/layered to feel dimensional
- White headline centered above screenshots
- Orange CTA button below
- Used for: Developer-targeted ads, product demos

### Pattern 3: Light Testimonial Grid (from community section)
- Off-white background (`#F5F1EB`)
- White cards with quotes in large serif/sans text
- Name + title + company below each quote
- Used for: Social proof ads, carousel slides

### Pattern 4: Hero Banner (from Agent page)
- White/light background
- Large bold headline centered
- Customer logos in a scrolling strip below
- Orange "Get started free" CTA pill
- Used for: Brand awareness, top-of-funnel

## Ad Format Specs

### Meta/Instagram Feed (4:5)
- Canvas: 1080 x 1350px
- Safe zone: 48px padding all sides
- Bottom CTA bar: 100-120px height

### Meta/Instagram Story (9:16)
- Canvas: 1080 x 1920px
- Top safe zone: 80px (profile/camera overlap)
- Bottom safe zone: 120px (swipe-up/CTA overlap)

### LinkedIn Feed (1.91:1)
- Canvas: 1200 x 628px
- Text overlay limited to top 60% (LinkedIn truncates)

### Twitter/X (16:9)
- Canvas: 1600 x 900px
- Key message in center 70%

## Voice in Visual Copy

- Headlines: Short, punchy, specific. Max 8 words.
- Body: One sentence max. Conversational, not corporate.
- Always use "builders" never "users"
- Proof points use exact numbers ("7,000 apps" not "thousands")
- CTA matches segment (see creative-dna.json)

## Don'ts

- No gradients on orange (flat only)
- No stock photography
- No generic "AI" imagery (robot icons, neural network graphics)
- No feature comparison tables in ads
- No more than 2 font weights per ad
- No text smaller than 14px (readability on mobile)
- Never use the word "revolutionary," "seamless," or "leverage"
