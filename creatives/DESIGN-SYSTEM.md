# Creative Factory -- Ad Design System

All ad mockups generated by `/creative-factory` must follow this design language.

**This is a template.** Run `/creative-factory init` to populate it with your brand's colors, fonts, and visual identity. See `examples/replit/DESIGN-SYSTEM.md` for a completed example.

---

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| `brand-primary` | | CTAs, logo, hero backgrounds, accent highlights |
| `dark-bg` | | Product UI backgrounds, dark-theme ads |
| `dark-surface` | | Cards, panels, elevated surfaces on dark |
| `dark-border` | | Borders, dividers on dark backgrounds |
| `light-bg` | | Marketing page backgrounds, testimonial sections |
| `text-primary` | `#FFFFFF` | Headings on dark, text on brand color |
| `text-secondary` | | Body copy on dark backgrounds |
| `text-muted` | | Captions, labels, secondary info on dark |
| `text-dark` | `#1A1A1A` | Headings on light/white backgrounds |
| `green-success` | `#28CA41` | Live badges, success states, security indicators |

## Typography

- **Font:** (set during init)
- **Headlines:** 700-900 weight, large (28-48px in ad units)
- **Body:** 400-500 weight, 16-20px
- **Captions/labels:** 500-600 weight, 12-14px, uppercase tracking for labels
- **Line height:** 1.3 for headlines, 1.5 for body

## Layout Patterns

### Pattern 1: Feature Panel
- Left: Brand primary color background with white headline + body text
- Right: Dark product screenshot with subtle drop shadow
- Used for: Feature callouts, capability highlights

### Pattern 2: Dark Product Showcase
- Full dark background
- Product screenshots in dark mode, overlapping/layered
- White headline centered above screenshots
- Brand-colored CTA button below
- Used for: Technical audience ads, product demos

### Pattern 3: Light Testimonial Grid
- Off-white background
- White cards with quotes
- Name + title + company below each quote
- Used for: Social proof ads, carousel slides

### Pattern 4: Hero Banner
- White/light background
- Large bold headline centered
- Customer logos in a scrolling strip below
- Brand-colored CTA pill
- Used for: Brand awareness, top-of-funnel

## Ad Format Specs

### Meta/Instagram Feed (4:5)
- Canvas: 1080 x 1350px
- Safe zone: 48px padding all sides
- Bottom CTA bar: 100-120px height

### Meta/Instagram Story (9:16)
- Canvas: 1080 x 1920px
- Top safe zone: 80px (profile/camera overlap)
- Bottom safe zone: 120px (swipe-up/CTA overlap)

### LinkedIn Feed (1.91:1)
- Canvas: 1200 x 628px
- Text overlay limited to top 60% (LinkedIn truncates)

### Twitter/X (16:9)
- Canvas: 1600 x 900px
- Key message in center 70%

## Voice in Visual Copy

- Headlines: Short, punchy, specific. Max 8 words.
- Body: One sentence max. Conversational, not corporate.
- Use vocabulary from config.json voice.say
- Avoid vocabulary from config.json voice.never_say
- Proof points use exact numbers (specifics beat generalities)
- CTA matches segment (see creative-dna.json)

## Don'ts

- No gradients on brand color (flat only)
- No stock photography
- No generic "AI" imagery (robot icons, neural network graphics)
- No feature comparison tables in ads
- No more than 2 font weights per ad
- No text smaller than 14px (readability on mobile)
- Avoid words listed in config.json voice.never_say
